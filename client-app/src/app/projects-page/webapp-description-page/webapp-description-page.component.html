Веб-приложение Geosystem

Веб-приложение Geosystem - это платформа для активных граждан, готовых обеспечивать актуальной информацией о состоянии дорожного покрытия федеральные, муниципальные и частные службы, занимающихся мониторингом и обслуживанием дорожного полотна

Ключевые возможности

Фотографирование дорожных дефектов

Получение бонусов за верифицированые фотографии

Просмотр карты с размеченными дефектами в доступных районах

Построение безопасного маршрута в обход опасных участков

Преимущества

Автоматическая проверка изображений с помощью технологии компьютерного зрения

Построение подробной графической карты с размеченными дорожными дефектами

Возможность построения безопасного маршрута в зависимости от состояния автодороги

Результаты опроса гражданских лиц
